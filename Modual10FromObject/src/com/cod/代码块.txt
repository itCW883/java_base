  构造代码块
  构造代码块每次都会优先于构造方法执行,new多少次对象就会执行多少次构造代码块
  格式:
       {
           方法体
       }

   静态代码块
   静态代码块优先于构造代码块和构造方法执行,但是只会执行一次,无论new多少对象都只会执行一次
   格式:
      static {
          方法体
      }
   最重要的使用场景:用于一些只需要执行一次且需要最先初始化的的数据
             例如:jdbc技术操作数据库( java连接和数据库四大参数
                                    1:注册驱动
                                    2:初始化操作数据库的地址
                                    3:初始化数据库的用户名
                                    4:初始化数据库密码      )
